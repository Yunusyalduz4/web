📱 UYGULAMA GENEL BİLGİLERİ
================================================================================
• Uygulama Adı: RANDEVUO
• Açıklama: Modern Kuaför Randevu ve Yönetim Sistemi
• Platform: PWA (Progressive Web App) - Mobil ve Desktop Uyumlu
• Dil: Türkçe
• Tema: Modern, Minimalist, Glassmorphism Tasarım
• Renk Paleti: Rose-Fuchsia-Indigo Gradient

🛠️ TEKNOLOJİ STACK
================================================================================
• Frontend: Next.js 15, React 18, TypeScript
• Styling: Tailwind CSS
• Backend: tRPC, PostgreSQL (Neon Database)
• Authentication: NextAuth.js
• Email Service: Resend
• Real-time: Socket.io, WebSocket
• Push Notifications: Web Push API, Service Workers
• Deployment: Vercel
• Database: PostgreSQL (Neon Cloud)
• Maps: Google Maps API
